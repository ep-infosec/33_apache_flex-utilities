Sample Name ---- Spark CurrencyFormatter

Introduction:
The CurrencyFormatter class provides locale-sensitive formatting and parsing of currency values. This class is a wrapper class around the flash.globalization.CurrencyFormatter. Therefore, the locale-specific formatting is provided by the flash.globalization.CurrencyFormatter. 

For more information about flash.globalization.CurrencyFormatter, please visit: http://help.adobe.com/en_US/FlashPlatform/beta/reference/actionscript/3/flash/globalization/CurrencyFormatter.html

Usage of the sample code:
1. Select a Locale from the locale comboBox first. It will use en_US as the default choice.
2. Enter a currency number in the text input.
3. The currency number will be formatted.
