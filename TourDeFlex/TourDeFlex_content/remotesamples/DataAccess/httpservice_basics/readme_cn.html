通过 HTTPService，可以向服务器发送 HTTP 请求，并使用响应。HTTP 和 HTTPS 都受支持。尽管 HTTPService 可以用于使用不同类型的响应，但其通常用于使用 XML。可以将 HTTPService 与以下任何服务器端技术结合使用：JSP、Servlet、ASP、Ruby on Rails、PHP 等等。可以在 HTTPService 的 url 属性中指定目标服务。

Flex 具有高级数据绑定功能。可以将某一属性的值与另一属性的值绑定，通常也可以与表达式绑定。在此示例中，DataGrid 的 dataProvider 属性与 HTTPService 的 lastResult 属性绑定（使用大括号表示法）。
