使用 RemoteObject，可以直接调用部署在应用程序服务器中的 Java 对象的方法，并使用返回值。返回值可以是基本数据类型的值、对象、对象集合、对象图形等等。

RemoteObject 的 destination 属性的值是在 remoting-config.xml 中映射到完全限定 java 类的逻辑名称。

通过服务器端方法返回的 Java 对象会反序列化为动态或类型 ActionScript 对象。在此示例中，没有提供 Product Java 类的显式 ActionScript 版本。因此，Product 对象将反序列化为动态对象。

与 HTTPService 和 WebService 类似，RemoteObject 调用也是异步调用。可以使用 RemoteObject 的 result 和 fault 事件处理结果和错误。
